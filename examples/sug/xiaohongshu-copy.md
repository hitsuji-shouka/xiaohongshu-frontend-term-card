## 正文/简介

搜索框里输入几个字，下面立刻冒出一排候选词，这就是前端常说的 SUG。🔍

📌 两张图讲清楚：
✅ SUG = 搜索建议 / 搜索联想，帮助用户少打字
✅ 输入 → 防抖 → 请求 → 召回排序 → 展示建议
✅ 它和 Select 的核心区别：SUG 支持自由输入，Select 只能从固定选项里选
⚠️ 实现时还要处理旧请求、键盘操作和中文输入法

💡 记忆口诀：SUG 是帮用户表达搜索意图，不是替用户做决定。
💾 前端术语入门，建议收藏！
